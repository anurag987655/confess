# Confess

A playful static web page with:
- A beating heart animation
- A dodging "No" button
- A "Yes" success screen with confetti

## Run locally

Because this is a static project, you can run it in either of these ways:

1. Open `index.html` directly in your browser.
2. Or serve it with a local server:
   - Python: `python3 -m http.server 8000`
   - Then open `http://localhost:8000`

## Project files

- `index.html` - page markup
- `style.css` - styles and animations
- `script.js` - button interaction and confetti logic
- `requirements.txt` - Python dependency info
